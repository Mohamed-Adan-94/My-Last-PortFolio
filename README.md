# 📁 Mohamed Adan Mohamud · Portfolio

> A professional, modern, and fully responsive personal portfolio website built with HTML, CSS, and Vanilla JavaScript.

## 📖 About The Project
This is a complete personal portfolio website for **Mohamed Adan Mohamud** — a Computer Networks & Security student at Jamhuriya University of Science & Technology. 
The portfolio showcases his skills, projects, education, and professional journey in networking, cybersecurity, web development, and leadership.

## ✨ Features

### 🎨 Design
- ✅ Clean, modern, and professional light theme
- ✅ Fully responsive (mobile, tablet, desktop)
- ✅ Smooth animations and transitions
- ✅ Professional typography with Google Fonts
- ✅ Font Awesome icons for visual elements

### 📄 Sections
- **Home** – Hero section with typing animation
- **About** – Biography, mission, vision, stats, core values
- **Education** – Academic background and certifications
- **Skills** – Technical skills with animated progress bars
- **Experience** – Leadership and involvement
- **Projects** – Portfolio of work with technology tags

### ⚡ Interactive Features

## 🚀 Getting Started

### Option 1: Open Directly
1. Download all files to your computer
2. Open `index.html` in your web browser
3. That's it! No server required.

### Option 2: Live Server (Recommended for development)
1. Install VS Code
2. Install the "Live Server" extension
3. Right-click on `index.html` → Open with Live Server
